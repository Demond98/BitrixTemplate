# BitrixTemplate
## Commands:
* npm run build - dev сборка проекта
* npm run watch - live dev сборка  проекта
* npm run prod - production сборка проекта и упаковка в zip-архив

## Структура проекта:
* dist/ - результат сборки
* src/
  * components/ - компоненты react
  * styles/ - стили
  * index.html
  * index.tsx
* .babelrc
* package.jso
* tsconfig.json
* tslint.json
* webpack.config.js
 
## Библиотеки:
* bootstrap
* jquery
* popper.js
* react
